# Archived — AssistanceKit Plans

These plans were created during the `AssistanceKit` era in the
`wichtelimwald/assistance` mono-repo and relate to the Markdown editor,
CoverFlow, and GlassOverlay modules that were **spun off into separate repos**
as part of ADR-0010:

| Spun-off module | Repo |
|-----------------|------|
| Markdown | `wichtelimwald/markdown-ui` |
| CoverFlow | `wichtelimwald/coverflow` |
| GlassOverlay | `wichtelimwald/glass-overlay` |

None of these plans are actionable in `wichtelimwald/shared-ui`, which only
ships the Backgrounds, Buttons, Compatibility, and Styles modules.

Archived: 2026-05-31
